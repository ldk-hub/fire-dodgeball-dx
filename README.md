# 🔥 불꽃의 피구왕 DX (Fire Dodgeball DX)

레트로 패미컴 감성의 HTML5 Canvas 피구 게임. 단일 `index.html` 파일, 외부 의존성 0개.

**▶ 플레이: https://ldk-hub.github.io/fire-dodgeball-dx/**

## 조작

| 키 | 동작 |
|---|---|
| `←→↑↓` | 이동 (같은 방향 두 번 = 대시) |
| `Z` | 던지기 (꾹 누르면 차지 → 풀차지 시 **필살슛**) / 수비 시 캐치 |
| `X` | 외야 패스 (협공) |
| `C` / `Space` | 점프 회피 (공중에서 Z = 점프슛) |
| `M` | BGM 토글 |
| `Enter` | 시작 / 일시정지 |

한/영 입력기 상태와 무관하게 동작합니다 (`e.code` 물리 키 판정).

## 특징

- **캐릭터별 고유 필살슛 8종** — 직선 화염 · 커브 · 지그재그 · 중량탄 · 급강하 궤적 물리
- **강도별 피격 리액션** — 강타에 맞으면 회전하며 날아가 바운드 후 다운
- **프레임 단위 도트 애니메이션** — 런 사이클, 와인드업, 캐치, KO, 승리 환호 + 스쿼시&스트레치
- **필살슛 컷인 연출** — 히트스톱 + 스피드라인 + 캐릭터 클로즈업
- **CRT 스캔라인 + 칩튠 BGM** (WebAudio) — 에셋 파일 없이 전부 코드 생성
- 라운드 승리 시 CPU 레벨 상승 (LV.3까지)

## 실행

브라우저에서 `index.html`을 열면 끝. 빌드·설치 불필요.
